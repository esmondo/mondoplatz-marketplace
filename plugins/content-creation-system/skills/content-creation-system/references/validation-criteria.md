# Validation Criteria for Content Ideas

## Dimensions (weighted)

| Dimension | Weight | Description |
|-----------|--------|-------------|
| Audience Relevance | 0.25 | How well the idea addresses audience needs |
| Uniqueness | 0.20 | Differentiation from existing content |
| Production Feasibility | 0.20 | Resources, time, technical requirements |
| Brand Alignment | 0.20 | Fit with brand voice, pillars, goals |
| Potential Impact | 0.15 | Expected reach, engagement, conversion |

## Scoring (1–5)

- 1: Reject — Does not meet minimum standards
- 2: Weak — Significant gaps
- 3: Acceptable — Meets baseline, consider with improvements
- 4: Strong — Good fit, recommended for production
- 5: Excellent — Exceptional fit, high priority

## Thresholds

- Minimum passing: 3.0
- Recommended: 4.0
- Automatic approval: 4.5

## Platform Fit

- **TikTok**: Hook (3s), trend-ready, audio-driven
- **IG**: Visual quality, 15–90s, editing complexity
- **YT**: Depth potential, 8–15min, retention structure
